## TTSがチェス学習を変える理由

アノテーション付きの棋譜を見直すとき、目は二重の作業をしています。盤上の駒を追い*ながら*、同時にコメントを読もうとします。視線が盤面とアノテーションパネルの間を行き来するたびに、一瞬だけポジションを見失います。駒を再確認し、ラインをたどり直し、頭の中でイメージを再構築しなければなりません。

テキスト読み上げ（TTS）はこの問題を完全に解決します。

TTSを有効にすると、棋譜を進めるとアノテーションが*読み上げられます*。目は盤面に集中できます。ナイトがf3に着地するのを見ながら、なぜそれが強い展開の手なのかを声が教えてくれます。盤面と言葉が同時に届きます。まるで目の前に座ったコーチが教えてくれるように。

これは特に以下の場面で効果的です：

- **オープニング学習** — ポジションが展開していくのを見ながら、各手の背後にあるアイデアを聞く
- **棋譜レビュー** — 自分のアノテーション付き棋譜を自然にレッスンを吸収しながら進める
- **エンドゲーム練習** — 重要なマスに集中しながら、コメントがガイドしてくれる
- **言語イマージョン** — フランス語、ドイツ語、スペイン語、ロシア語、日本語、中国語、韓国語でチェスを学習。すべてのチェス用語が翻訳されます
- **アクセシビリティ** — 読むより聞く方が楽な方、デスクから離れて学習したい方に

一度試すと、音声なしのアノテーションに戻るのは、映画をミュートで見るようなものに感じるでしょう。

## プロバイダーを選ぶ

En Parlant~には5つのTTSプロバイダーが搭載されています。始めるのに必要なのは1つだけです。

### ElevenLabs

最高の音声品質。ElevenLabsは表現豊かで人間のような音声を生成します。数十の個性的な声から選べます。8言語すべてに対応し、CJKの発音も優秀です。

無料枠は月10,000文字（アノテーション付き棋譜2～5局分）。有料プランは月5ドルから。セットアップは簡単：アカウント作成、APIキーをコピー、En Parlant~に貼り付け。

インターネット接続が必要。音声品質にこだわる方に最適。

**[ElevenLabsセットアップガイド](docs/tts/setup-elevenlabs.md)**（英語）

### Google Cloud TTS

品質、言語サポート、コストの最良のバランス。GoogleのWaveNetニューラル音声は8言語すべてで自然でクリアです。無料枠は寛大で、月100万文字で数百局の棋譜をカバーできます。

セットアップは約5分。無料枠を超えない限り課金されません。

インターネット接続が必要。ほとんどのユーザーに最適。

**[Google Cloudセットアップガイド](docs/tts/setup-google.md)**（英語）

### KittenTTS

完全にローカルで動作する高品質AIです。約25MBの軽量ニューラルモデルを使用し、8つの表現豊かな音声（男性4、女性4）を提供します。品質は驚くほど優秀です。

ハードウェアが必要：KittenTTSはPyTorchを使用してCPU推論を行うため、モダンなマルチコアプロセッサが必要です。現在は英語のみ。

インターネット不要。APIキー不要。最高のローカル品質。

**[KittenTTSセットアップガイド](docs/tts/setup-kittentts.md)**（英語）

### System TTS

OSに内蔵された音声合成。インストール不要、APIキー不要、サーバー不要。選択するだけで即座に動作します。音声品質は基本的ですが、セットアップ不要で即座に機能します。

Linuxではepeakまたはspeech-dispatcher、macOSではシステム音声、WindowsではSAPIです。

インターネット不要。クイックテストに最適。

**[System TTSセットアップガイド](docs/tts/setup-system.md)**（英語）

### OpenTTS

Dockerでマシン上で動作するオープンソースTTSサーバー。データはコンピューターの外に出ません。複数のTTSエンジン（Larynx、Festival、eSpeak、Coqui-TTS）をバンドルし、英語だけで75以上の音声を提供します。

品質面では旧世代のエンジンのため、ElevenLabsやGoogleよりロボット的です。ヨーロッパ言語で最も良く動作します。CJKは十分にサポートされていません。将来のリリースで削除される可能性があります。

インターネット不要。APIキー不要。最大限のプライバシーに最適。

**[OpenTTSセットアップガイド](docs/tts/setup-opentts.md)**（英語）

### おすすめ

最高の音声品質なら**ElevenLabs**から始めてください。品質と無料利用のベストバランスなら**Google Cloud**が月数百局をカバーします。高品質ローカルTTSなら**KittenTTS**がモダンCPUで優秀です。設定なしのテストなら**System TTS**が即座に動作します。最大限のプライバシーなら**OpenTTS**がDockerでローカル動作します。

## 設定リファレンス

すべてのTTS設定は**Settings > Sound**にあります：

| 設定 | 機能 |
|------|------|
| **Text-to-Speech** | すべてのTTS機能のメインスイッチ |
| **Auto-Narrate on Move** | 手を進めるとアノテーションを自動的に読み上げ |
| **TTS Provider** | 5つのプロバイダーを切り替え |
| **TTS Voice** | プロバイダー固有の音声選択 |
| **TTS Language** | ナレーション言語 — チェス用語は自動翻訳 |
| **TTS Volume** | ナレーションの音量 |
| **TTS Speed** | 再生速度（0.5倍〜2倍）— オーディオ再生成なしで調整 |
| **ElevenLabs API Key** | ElevenLabs APIキー |
| **Google Cloud API Key** | Google Cloud APIキー |
| **KittenTTS CPU Threads** | 推論用CPUスレッド（0 = 自動） |
| **TTS Audio Cache** | オーディオキャッシュをクリアして再生成を強制 |

## サポートされている言語

TTSナレーションは8言語に対応し、チェス用語が完全に翻訳されています：

| 言語 | チェスの例 |
|------|----------|
| **English** | Knight f3, check. A strong developing move. |
| **Francais** | Cavalier f3, echec. Un coup de developpement fort. |
| **Espanol** | Caballo f3, jaque. Un fuerte movimiento. |
| **Deutsch** | Springer f3, Schach. Ein starker Entwicklungszug. |
| **日本語** | ナイト f3、チェック。強い展開の手。 |
| **Русский** | Конь f3, шах. Сильный развивающий ход. |
| **中文** | 马 f3，将军。一步控制中心的强力出子。 |
| **한국어** | 나이트 f3, 체크. 중앙을 지배하는 강력한 전개 수. |

すべてのチェス用語 — 駒の名前、「チェック」「チェックメイト」「キャスリング」「テイクス」、「ブリリアントムーブ」「ブランダー」などの評価 — は選択した言語で読み上げられます。

## より良い体験のためのヒント

- **Auto-Narrateを使いましょう。**「Auto-Narrate on Move」をオンにして矢印キーで棋譜を進めるだけです。コメントが自然に流れてきます。

- **自分の棋譜にアノテーションを付けましょう。** TTSが真価を発揮するのは*自分の*棋譜のコメントを聞くときです。

- **速度を変えてみましょう。** じっくり学習するなら1倍速、素早くレビューするなら1.3倍速。

- **スピーカーアイコンを使いましょう。** 各コメントに小さなスピーカーアイコンがあります。クリックするとそのアノテーションだけ聞けます。

- **言語を変えてチェス用語を学びましょう。** 第二言語でチェスを学んでいるなら、TTS言語を合わせてみてください。
