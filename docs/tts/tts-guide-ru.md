## Почему TTS меняет подход к изучению шахмат

Когда вы просматриваете аннотированную партию, ваши глаза выполняют двойную работу. Вы пытаетесь следить за фигурами на доске *и* одновременно читать комментарии. Ваш взгляд мечется между доской и панелью аннотаций, и каждый раз вы теряете позицию на долю секунды.

Text-to-speech полностью решает эту проблему.

С включённым TTS вы проходите по партии, и аннотации *читаются вслух*. Ваши глаза остаются на доске. Вы наблюдаете, как конь садится на f3, в то время как голос объясняет, почему это сильный развивающий ход. Доска и слова приходят вместе, как тренер, сидящий напротив вас.

Это особенно полезно для:

- **Изучение дебютов** — слушайте идеи за каждым ходом, наблюдая за развитием позиции
- **Обзор партий** — проходите по аннотированным партиям и усваивайте уроки естественно
- **Практика эндшпилей** — держите фокус на критических полях, пока комментарий вас направляет
- **Языковое погружение** — изучайте шахматы на французском, немецком, испанском, русском, японском, китайском или корейском с полным переводом терминов
- **Доступность** — для игроков, которым легче слушать, чем читать

Попробовав однажды, возвращение к безмолвным аннотациям будет как просмотр фильма без звука.

## Выбор провайдера

En Parlant~ поставляется с пятью провайдерами TTS. Вам нужен только один для начала.

### ElevenLabs

Лучшее качество голоса. ElevenLabs создаёт выразительную, человекоподобную речь с настоящей индивидуальностью. Десятки уникальных голосов. Поддержка всех восьми языков с отличным произношением CJK.

Бесплатный лимит — 10 000 символов в месяц (достаточно для 2-5 аннотированных партий). Платные планы от 5$/месяц за 30 000 символов. Простая настройка: создать аккаунт, скопировать API-ключ, вставить в En Parlant~.

Требуется интернет. Идеально для ценителей качества голоса.

**[Руководство по настройке ElevenLabs](setup-elevenlabs.md)** *(на английском)*

### Google Cloud TTS

Лучший баланс качества, языковой поддержки и стоимости. Нейронные голоса WaveNet от Google звучат естественно и чисто на всех восьми языках. Бесплатный лимит щедрый — один миллион символов в месяц покрывает сотни аннотированных партий.

Настройка занимает около 5 минут. Без оплаты, пока не превысите бесплатный лимит.

Требуется интернет. Идеально для большинства пользователей.

**[Руководство по настройке Google Cloud](setup-google.md)** *(на английском)*

### KittenTTS

Высококачественный локальный ИИ, работающий полностью на вашей машине. Использует лёгкую нейронную модель ~25 МБ с 8 выразительными голосами (4 мужских, 4 женских). Качество удивительно хорошее.

Компромисс — оборудование: KittenTTS использует PyTorch для инференса на CPU и требует современный многоядерный процессор. Пока только английский.

Интернет не требуется. Без API-ключа. Лучшее локальное качество.

**[Руководство по настройке KittenTTS](setup-kittentts.md)** *(на английском)*

### System TTS

Встроенный синтез речи вашей операционной системы. Ничего устанавливать не нужно, ни API-ключей, ни серверов. Выберите и вперёд. Качество голоса базовое, но работает мгновенно.

В Linux это обычно eSpeak или speech-dispatcher; в macOS — системный голос; в Windows — SAPI.

Интернет не требуется. Идеально для быстрого тестирования.

**[Руководство по настройке System TTS](setup-system.md)** *(на английском)*

### OpenTTS

Открытый TTS-сервер, работающий на вашей машине через Docker. Ничто не покидает ваш компьютер. Объединяет несколько TTS-движков (Larynx, Festival, eSpeak, Coqui-TTS), предлагая более 75 голосов только для английского.

Компромисс — качество: старые движки, поэтому звучит более роботизированно. Лучше всего работает с европейскими языками — CJK поддерживается плохо. OpenTTS может быть удалён в будущих версиях.

Интернет не требуется. Без API-ключа. Идеально для максимальной конфиденциальности.

**[Руководство по настройке OpenTTS](setup-opentts.md)** *(на английском)*

### Наша рекомендация

Начните с **ElevenLabs** для лучшего качества голоса. Для лучшего баланса качества и бесплатного использования **Google Cloud** покрывает сотни партий в месяц. Для высококачественного локального TTS **KittenTTS** отлично работает на современном CPU. Для тестирования без настройки **System TTS** работает мгновенно. Для максимальной конфиденциальности **OpenTTS** работает локально через Docker.

## Справочник настроек

Все настройки TTS находятся в **Settings > Sound**:

| Настройка | Функция |
|-----------|---------|
| **Text-to-Speech** | Главный переключатель для всех функций TTS |
| **Auto-Narrate on Move** | Автоматически озвучивать аннотации при переходе между ходами |
| **TTS Provider** | Переключение между пятью провайдерами |
| **TTS Voice** | Выбор голоса для провайдера |
| **TTS Language** | Язык озвучивания — шахматные термины переводятся автоматически |
| **TTS Volume** | Громкость озвучивания |
| **TTS Speed** | Скорость воспроизведения (0,5x до 2x) — регулируется без перегенерации аудио |
| **ElevenLabs API Key** | Ваш API-ключ ElevenLabs |
| **Google Cloud API Key** | Ваш API-ключ Google Cloud |
| **KittenTTS CPU Threads** | Потоки CPU для инференса (0 = авто) |
| **TTS Audio Cache** | Очистить кэш аудио для принудительной перегенерации |

## Поддерживаемые языки

TTS-озвучивание поддерживает восемь языков с полностью переведённой шахматной терминологией:

| Язык | Пример шахмат |
|------|--------------|
| **English** | Knight f3, check. A strong developing move. |
| **Francais** | Cavalier f3, echec. Un coup de developpement fort. |
| **Espanol** | Caballo f3, jaque. Un fuerte movimiento. |
| **Deutsch** | Springer f3, Schach. Ein starker Entwicklungszug. |
| **日本語** | ナイト f3、チェック。強い展開の手。 |
| **Русский** | Конь f3, шах. Сильный развивающий ход. |
| **中文** | 马 f3，将军。一步控制中心的强力出子。 |
| **한국어** | 나이트 f3, 체크. 중앙을 지배하는 강력한 전개 수. |

Все шахматные термины — названия фигур, «шах», «мат», «рокировка», «берёт», оценки ходов — произносятся на выбранном языке.

## Советы для лучшего опыта

- **Используйте Auto-Narrate.** Включите «Auto-Narrate on Move» и просто используйте стрелки для прохождения партий.

- **Комментируйте свои партии.** TTS по-настоящему раскрывается, когда вы слушаете комментарии к *своим* партиям.

- **Попробуйте разные скорости.** Для внимательного изучения — 1x, для быстрого обзора — 1,3x.

- **Используйте значок динамика.** У каждого комментария есть маленький значок динамика. Нажмите, чтобы услышать эту аннотацию.

- **Меняйте язык для изучения терминологии.** Если вы изучаете шахматы на втором языке, установите соответствующий язык TTS.
