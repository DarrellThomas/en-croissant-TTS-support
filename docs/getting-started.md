# Getting Started

This guide will help you get up and running with En Croissant-TTS Support.

## Installation

En Croissant-TTS Support supports Windows, Linux, and macOS.

### Windows

1. Visit the Download page
2. Select the Windows download button for the latest `.exe` installer
3. Execute the installer and complete on-screen prompts

### Linux

1. Access the Download page
2. Choose either `.AppImage` or `.deb` file based on your distribution
3. For AppImage:
   ```
   chmod +x en-croissant_amd64.AppImage
   ./en-croissant_amd64.AppImage
   ```
4. For .deb:
   ```
   sudo dpkg -i en-croissant_amd64.deb
   ```

### macOS

1. Go to the Download page
2. Download the `.dmg` file
3. Open the `.dmg` and transfer the En Croissant-TTS Support app to Applications

## First Run

On initial launch you'll see the main dashboard. Recommended setup steps:

1. **Configure Engines** — Visit the Engines page to download chess engines like Stockfish
2. **Add Reference Database** — Access the Databases page to download analysis reference materials
3. **Explore Features** — Try the Analysis board, play against the computer, or practice tactics
