## TTS가 체스 학습을 바꾸는 이유

주석이 달린 기보를 복습할 때, 여러분의 눈은 두 가지 일을 동시에 합니다. 보드 위의 기물을 따라가면서 동시에 해설도 읽어야 합니다. 시선은 보드와 주석 패널 사이를 오가며, 그때마다 잠시 국면을 놓치게 됩니다. 기물의 위치를 다시 찾고, 수순을 다시 따라가고, 머릿속에서 국면을 다시 구성해야 합니다.

텍스트 음성 변환 기능은 이 문제를 완전히 해결합니다.

TTS를 활성화하면, 기보를 넘기기만 해도 주석이 *음성으로 읽어* 줍니다. 눈은 보드에 집중한 채로 있을 수 있습니다. 나이트가 f3에 착지하는 것을 보면서 왜 그것이 뛰어난 전개 수인지를 음성으로 들을 수 있습니다. 폰 구조가 변하는 것을 보면서 그 뒤에 있는 전략적 아이디어를 해설이 설명해 줍니다. 보드와 말이 동시에 도착합니다 — 마치 맞은편에 코치가 앉아서 가르쳐 주는 것 같은 느낌입니다.

특히 다음과 같은 상황에서 효과적입니다:

- **오프닝 연구** — 국면이 전개되는 것을 보면서 각 수 뒤에 있는 아이디어를 귀로 들을 수 있습니다
- **기보 복습** — 자신의 주석이 달린 기보를 스텝 재생하며 교훈을 자연스럽게 흡수할 수 있습니다
- **엔드게임 연습** — 중요한 칸에 집중한 채로 해설이 안내해 줍니다
- **언어 학습과의 융합** — 프랑스어, 독일어, 스페인어, 러시아어, 일본어, 중국어, 한국어로 체스를 공부할 수 있습니다. 모든 체스 용어가 자동으로 번역됩니다. "Knight f3, check" 대신 "나이트 f3, 체크"라고 들립니다. 자신이 생각하는 언어로 체스를 배우세요.
- **접근성** — 읽는 것보다 듣는 것이 편한 분이나 책상을 떠나 학습하고 싶은 분에게 최적입니다

한번 사용해 보면, 음성 없는 주석으로 돌아가는 것은 영화를 무음으로 보는 것 같은 느낌이 될 것입니다.

## 프로바이더 선택

이 버전에는 Google Cloud와 ElevenLabs 두 가지 TTS 프로바이더가 탑재되어 있습니다. 커뮤니티에서 다른 프로바이더를 원하시면 알려주세요. 추가할 수 있습니다. 시작하는 데 필요한 프로바이더는 하나뿐입니다. 자신에게 맞는 것을 선택하세요.

|                        | Google Cloud                  | ElevenLabs                    |
|------------------------|-------------------------------|-------------------------------|
| **무료 한도**           | 월 100만 글자                  | 월 10,000 글자                 |
| **음성 품질**           | 매우 우수 (WaveNet)            | 탁월 (프리미엄 AI 음성)         |
| **음성 선택**           | 언어별 남성 또는 여성            | 수십 가지 고유한 캐릭터          |
| **유료 플랜**           | 종량제 (소액)                   | 월 $5부터                      |
| **추천 대상**           | 대부분의 사용자                 | 음성 품질을 중시하는 분          |
| **설정 난이도**         | 보통 (Cloud Console)           | 쉬움 (간단한 가입)              |

**추천:** **Google Cloud**부터 시작하세요. 무료 한도로 월 100만 글자까지 사용할 수 있습니다. 이는 수백 개의 완전히 주석이 달린 기보에 해당하며, 무료입니다. WaveNet 음성도 훌륭합니다. 나중에 더 풍부하고 표현력 있는 내레이션을 원하게 되면 ElevenLabs를 사용해 보세요.

## Google Cloud TTS 설정

이 과정은 약 5분 정도 걸립니다. Google 계정(Gmail이나 YouTube에서 사용하는 것)이 필요합니다.

### 1단계: Google Cloud Console에 로그인

1. 브라우저를 열고 **[console.cloud.google.com](https://console.cloud.google.com/)** 으로 이동합니다
2. Google 계정으로 로그인합니다
3. 처음인 경우 서비스 약관에 동의해야 합니다. 체크박스를 선택하고 **Agree and Continue**를 클릭합니다

Google Cloud Console 대시보드가 표시됩니다. 화면이 복잡해 보이지만, 여기서 필요한 것은 두 가지뿐입니다.

### 2단계: 결제 설정

Google은 무료 한도라도 결제 계정을 요구합니다. **월 100만 글자를 초과하지 않는 한 요금이 부과되지 않습니다** (체스 주석으로 그 정도를 사용하기는 매우 어렵습니다). 한도에 가까워지면 Google이 미리 경고를 표시합니다.

1. 상단 검색창에 **"Billing"**을 입력하고 드롭다운에서 **Billing**을 클릭합니다
2. **Link a billing account**를 클릭합니다 (처음이면 **Create account**)
3. 안내에 따라 신용카드 또는 직불카드를 등록합니다
4. 완료되면 결제 계정 옆에 녹색 체크 표시가 나타납니다

> **참고:** 다른 프로젝트에서 이미 Google Cloud 결제를 설정한 경우 이 단계를 건너뛸 수 있습니다. 기존 결제 계정을 그대로 사용할 수 있습니다.

### 3단계: Text-to-Speech API 활성화

사용할 서비스를 Google에 알려주는 단계입니다.

1. 상단 검색창에 **"Text-to-Speech"**를 입력합니다
2. 드롭다운 결과에서 **Cloud Text-to-Speech API** (파란색 API 아이콘이 표시)를 클릭합니다
3. API 세부 정보 페이지가 열립니다. 큰 파란색 **Enable** 버튼을 클릭합니다
4. 몇 초 기다립니다. 버튼이 **Manage**로 바뀌면 API가 활성화된 것입니다

### 4단계: API 키 생성

API 키는 En Croissant-TTS가 Google 서버와 통신하기 위해 사용하는 것입니다.

1. 상단 검색창에 **"Credentials"**를 입력하고 "APIs & Services" 아래의 **Credentials**를 클릭합니다
2. 페이지 상단 근처의 **+ Create Credentials**를 클릭합니다
3. 드롭다운에서 **API key**를 선택합니다
4. 다이얼로그에 새 키가 표시됩니다. `AIzaSyC...약 35자...`와 같은 형태입니다
5. 키 옆의 **복사 아이콘**을 클릭하여 클립보드에 복사합니다
6. **Close**를 클릭합니다

> **권장: 키를 제한하세요.** 키를 생성하면 Credentials 페이지에 목록으로 표시됩니다. 키 이름을 클릭하여 설정을 엽니다. **API restrictions** 항목에서 **Restrict key**를 선택한 다음 드롭다운에서 **Cloud Text-to-Speech API**를 선택하고 **Save**를 클릭합니다. 이렇게 하면 키가 유출되더라도 TTS 이외의 용도로는 사용할 수 없습니다.

### 5단계: En Croissant-TTS 설정

거의 다 되었습니다!

1. En Croissant-TTS를 열고 **Settings** (톱니바퀴 아이콘) > **Sound** 탭으로 이동합니다
2. TTS 섹션까지 스크롤합니다
3. **TTS Provider**를 **Google Cloud**로 설정합니다
4. **Google Cloud API Key** 필드를 클릭하고 키를 붙여넣습니다 (Ctrl+V)
5. **Text-to-Speech**를 **On**으로 설정합니다
6. 음성 선택 옆의 **Test** 버튼을 클릭합니다

체스 수가 음성으로 읽어지면 설정 완료입니다!

> **문제 해결:** 테스트 음성이 나오지 않으면 다음을 확인하세요. (1) API 키가 완전히 붙여넣어졌는지, (2) Text-to-Speech API가 활성화되었는지 (3단계), (3) 결제가 연결되었는지 (2단계). 가장 흔한 원인은 API 활성화를 잊은 것입니다.

## ElevenLabs 설정

ElevenLabs는 설정이 더 간단하지만 무료 한도가 작습니다 (월 10,000 글자 — 몇 게임을 시도해 보기에는 충분합니다).

### 1단계: 계정 생성

1. 브라우저를 열고 **[elevenlabs.io](https://elevenlabs.io/)** 로 이동합니다
2. 오른쪽 상단의 **Sign Up**을 클릭합니다
3. Google, GitHub 또는 이메일로 가입할 수 있습니다. 가장 편한 방법을 선택하세요
4. 가입이 완료되면 ElevenLabs 대시보드가 표시됩니다

### 2단계: API 키 받기

1. 대시보드 왼쪽 하단의 **프로필 아이콘** (또는 이름)을 클릭합니다
2. **Profile + API key**를 클릭합니다
3. API 키 섹션이 표시됩니다. **Reveal**을 클릭하여 키를 표시하거나, 아직 키가 없으면 **Generate**를 클릭합니다
4. 키는 `sk_...약 30자...`와 같은 형태입니다
5. **복사 아이콘**을 클릭하여 클립보드에 복사합니다

### 3단계: En Croissant-TTS 설정

1. En Croissant-TTS를 열고 **Settings** (톱니바퀴 아이콘) > **Sound** 탭으로 이동합니다
2. TTS 섹션까지 스크롤합니다
3. **TTS Provider**를 **ElevenLabs**로 설정합니다
4. **ElevenLabs API Key** 필드를 클릭하고 키를 붙여넣습니다 (Ctrl+V)
5. **TTS Voice** 드롭다운에 사용 가능한 음성이 표시됩니다. **Adam**은 훌륭한 기본 선택입니다 — 명확하고 자연스러우며 체스 해설에 잘 어울립니다
6. **Text-to-Speech**를 **On**으로 설정합니다
7. 음성 선택 옆의 **Test** 버튼을 클릭합니다

체스 수가 음성으로 읽어져야 합니다.

> **무료 한도에 대해:** ElevenLabs 무료 플랜에서는 월 10,000 글자까지 사용할 수 있습니다. 주석이 달린 기보 1건당 약 2,000~4,000 글자를 사용하므로 매월 2~5개의 기보를 무료로 복습할 수 있습니다. TTS가 유용하다고 느끼시면 Starter 플랜 (월 $5, 30,000 글자)을 추천합니다. Pro 플랜 (월 $22, 100,000 글자)은 헤비 사용자를 위한 것입니다.

## 설정 참조

모든 TTS 설정은 **Settings > Sound**에 있습니다:

| 설정 항목                  | 기능                                                                           |
|--------------------------|-------------------------------------------------------------------------------|
| **Text-to-Speech**       | 모든 TTS 기능의 메인 온/오프 스위치                                                |
| **Auto-Narrate on Move** | 화살표 키로 수를 넘길 때 주석을 자동으로 읽어 줍니다                                  |
| **TTS Provider**         | ElevenLabs와 Google Cloud 간 전환                                               |
| **ElevenLabs API Key**   | ElevenLabs API 키 (ElevenLabs를 사용하는 경우에만 필요)                            |
| **Google Cloud API Key** | Google Cloud API 키 (Google을 사용하는 경우에만 필요)                               |
| **TTS Voice**            | ElevenLabs: 원하는 음성 선택. Google: 남성 또는 여성 선택                           |
| **TTS Language**         | 내레이션 언어 — 모든 체스 용어가 자동으로 번역됩니다                                  |
| **TTS Volume**           | 내레이션 볼륨                                                                    |
| **TTS Speed**            | 재생 속도 (0.5배~2배) — 오디오 재생성 없이 즉시 조정됩니다                            |
| **TTS Audio Cache**      | 캐시된 오디오를 지워 재생성을 강제합니다 (주석 편집 후 유용합니다)                      |

## 지원 언어

TTS 내레이션은 현재 체스 용어가 완전히 번역된 8개 언어를 지원합니다:

| 언어                | 체스 예시                                             |
|--------------------|------------------------------------------------------|
| **English**        | Knight f3, check. A strong developing move.          |
| **Français**       | Cavalier f3, échec. Un coup de développement fort.   |
| **Español**        | Caballo f3, jaque. Un fuerte movimiento.             |
| **Deutsch**        | Springer f3, Schach. Ein starker Entwicklungszug.    |
| **日本語**          | ナイト f3、チェック。強い展開の手。                       |
| **Русский**        | Конь f3, шах. Сильный развивающий ход.               |
| **中文**            | 马 f3，将军。一步控制中心的强力出子。                     |
| **한국어**          | 나이트 f3, 체크. 중앙을 지배하는 강력한 전개 수.          |

기물 이름, "체크", "체크메이트", "캐슬링", "잡음", 그리고 "브릴리언트한 수"와 "대실수" 같은 수의 평가 — 모든 체스 용어가 선택한 언어로 읽어집니다. PGN 파일의 코멘트는 그대로 읽어지므로, 듣고 싶은 언어로 기보에 주석을 달아 주세요.

## 더 나은 경험을 위한 팁

- **Auto-Narrate를 활용하세요.** "Auto-Narrate on Move"를 켜고 화살표 키로 기보를 넘기기만 하면 됩니다. 수를 넘길 때마다 해설이 자연스럽게 흘러나옵니다. 마치 어깨 너머로 코치가 있는 것 같은 느낌입니다.

- **자신의 대국에 주석을 달아 보세요.** TTS가 진가를 발휘하는 것은 *자신의* 대국 해설을 들을 때입니다. 대국에 주석을 단 후 내레이션과 함께 스텝 재생해 보세요. 국면을 바라보면서 "이 폰을 잡는 것은 매력적으로 보이지만, 킹사이드는 아직 전혀 전개되지 않았습니다"라고 들려오는 경험은 텍스트로 읽는 것과는 전혀 다릅니다.

- **재생 속도를 바꿔 보세요.** 꼼꼼히 연구하려면 1배속, 빠르게 복습하려면 1.3배속을 선호하는 분이 많습니다. 속도 슬라이더는 실시간으로 재생 속도를 조절하며 추가 API 글자 수를 소비하지 않습니다. 오디오는 한 번만 생성되고 재생 속도만 바뀝니다.

- **스피커 아이콘을 활용하세요.** 수순 목록의 각 코멘트에는 작은 스피커 아이콘이 있습니다. 클릭하면 전체 기보를 넘기지 않고 해당 주석만 들을 수 있습니다.

- **언어를 바꿔서 체스 용어를 배우세요.** 제2언어로 체스를 공부하고 있다면 TTS 언어를 맞춰 보세요. 듣기만 해도 "Cavalier" (나이트), "échec" (체크), "mat" (체크메이트) 같은 용어가 자연스럽게 익혀집니다.

## 이 기능에 대해

En Croissant는 [Francisco Salgueiro](https://github.com/franciscoBSalgueiro)가 개발한 오픈소스 체스 학습 도구입니다. Francisco는 정말로 뛰어난 것을 만들었습니다. 무료이고 강력한, 커뮤니티 중심의 체스 학습 플랫폼을 GPL-3.0 라이선스로 공개하여 누구나 사용하고 개선하고 공유할 수 있게 했습니다. 이 TTS 기능이 존재하는 것은 그의 관대함 덕분입니다. 그가 구축한 기반에 감사하며, 이 프로젝트에 기여할 수 있음을 자랑스럽게 생각합니다.

TTS 플러그인은 [Red Shed](mailto:darrell@redshed.ai)의 Darrell이 [Claude Code](https://claude.ai/claude-code)의 도움을 받아 개발했습니다. 다국어 지원, 듀얼 프로바이더 통합, 8개 언어에 걸친 체스 용어 번역 — 모두 소스에서 빌드하고, 직접 테스트하고, 정성을 담아 기여했습니다.
